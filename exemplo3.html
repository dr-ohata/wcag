Problemas de Foco Visível ao Navegar por Teclado

Esta página demonstra problemas comuns relacionados ao indicador de foco durante a navegação por teclado.

Importante: Experimente navegar por esta página usando apenas o teclado (tecla Tab para avançar, Shift+Tab para voltar).

1. Indicador de Foco Removido ou Desativado

Problema: Indicador de Foco Removido com CSS

Muitos sites removem o outline padrão dos elementos focados, tornando impossível saber onde o usuário está durante a navegação por teclado.

Experimente navegar por estes botões usando o teclado:

/* CSS problemático - NUNCA faça isso */
.no-focus:focus {
  outline: none;
}

Solução: Manter ou Melhorar o Indicador de Foco

O indicador de foco deve ser sempre visível e claro. Você pode personalizá-lo para combinar com o design do seu site, mas nunca removê-lo completamente.

Experimente navegar por estes botões usando o teclado:

/* CSS acessível - personalize mas mantenha visível */
.good-button:focus {
  outline: 3px solid #80bdff;
  outline-offset: 2px;
}

2. Indicador de Foco com Baixo Contraste

Problema: Indicador de Foco com Cores de Baixo Contraste

Às vezes o indicador de foco existe, mas tem contraste insuficiente com o fundo, tornando-o difícil de ver.

Experimente navegar por este botão usando o teclado:

/* CSS problemático - contraste insuficiente */
.bad-button:focus {
  outline: 2px solid #aaaaaa; /* Cinza claro */
}

Solução: Indicador de Foco com Alto Contraste

O indicador de foco deve ter contraste suficiente com a cor de fundo para ser claramente visível.

Experimente navegar por este botão usando o teclado:

/* CSS acessível - alto contraste */
.good-button:focus {
  outline: 3px solid #0069d9; /* Azul escuro */
  outline-offset: 2px;
}

3. Navegação em Menus

Problema: Itens de Menu Sem Indicador de Foco

Menus de navegação frequentemente não possuem indicadores de foco adequados, tornando difícil saber qual item está selecionado.

  • Contato

Solução: Itens de Menu com Foco Visível

Os itens de menu devem ter um indicador de foco claro e distinto.

Experimente navegar por este menu usando o teclado:

  • Início
  • Produtos
  • Serviços
  • Contato
/* CSS acessível para itens de menu */
.nav-item:focus {
  background-color: #007bff;
  outline: 2px solid #80bdff;
  outline-offset: -2px;
}
